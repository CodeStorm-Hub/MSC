1. Executive Summary

1.1 Purpose & Scope

This Software Requirements Specification (SRS) defines comprehensive, implementable requirements for MSC Home Rental & Real Estate, a verified property marketplace for Bangladesh that includes communication, legal & financial support, secure payments, and reputation-based trust.

MVP Scope Includes:

  • Account creation + professional mode switching
  • Identity + professional verification (badges)
  • Verified property listings + listing Accuracy Score
  • Advanced search + map-based search + favorites
  • In-platform chat + audio/video + appointment booking
  • Offers/negotiation ‚Üí order ‚Üí payment ‚Üí transaction tracking
  • Legal support directory + booking/case tracking
  • Financial/loan support directory + requests
  • Post-transaction ratings/reviews

Deferred to P2:

  • Community features (groups/pages/posts)
  • Blogs/videos publishing (moderated)

3.3 Roles & Permissions (RBAC)

Access Control Model

MSC Home uses a hybrid access control approach:

  • RBAC (Role-Based Access Control) for coarse permissions (what a role may do)
  • ABAC (Attribute-Based Access Control) for fine permissions (what a user may do to a specific resource)

Role Catalog

Guest

Not logged in. Can browse public listings only.

User (Consumer)

Logged in buyer/renter/seller (non-professional). Can create listings, communicate, make offers.

Professional

Verified or unverified professionals: Agent/Realtor, Developer/Company, Legal Agent, Financial Agent, Service Provider.

Verifier / Moderator

Reviews identity/listing verification and moderation cases.

Customer Support

Assists in disputes and user issues (limited admin powers).

Admin

Full platform management and system administration.
